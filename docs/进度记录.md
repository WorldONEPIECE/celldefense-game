## T13 实现内容

**新建：** `src/js/map-system.js`

**设计：** 5层地图，第1-4层每层2个可选节点（病毒随机分配），第5层固定HIV（自动选择）。层间交叉路线（layer i 的节点B = layer i+1 的节点A），制造分支感。

**关键bug修复：** `Object.values(levelsData)` 会取出 `_comment` 字符串，导致 `i18n.get(undefined)` 崩溃。过滤条件：`l && typeof l === 'object' && l.virus_id`。

**流程：** 胜利 → 奖励卡选择（若有）→ 地图弹出 → 选节点 → `startNextLevel(levelId)`。HIV最终关（next_level=null）直接 `showGameOver(true)`，跳过地图。

## T14 实现内容

**改动：** `ui-renderer.js` v1.2、`zh-CN.json`（新增 `pathways.*`）、`index.html`（新增 `.pathway-suppressed` CSS）

**功能：**
- `renderActivePathways()`：读 `GameState.activePathways` 和 `suppressedPathways`，渲染为彩色标签（绿=活跃，红⛔=被压制）
- `renderDefenseLayers()`：`PATHWAY_LAYER_MAP` 定义通路→防线层映射，有效通路（未被压制）激活对应层高亮
- `renderAll()` 中新增调用两者
