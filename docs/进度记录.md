## 重要技术决策（写新代码前必须知道）

**tickStatuses 在病毒回合开头调用：** 病毒本回合加入的 suppressedPathways remaining+1（tick后=原值），下一玩家回合时有效，结束时再tick掉。

**永久持续状态用 Infinity：** `remaining_turns: Infinity`，`Infinity-1===Infinity`，不会被 filter 移除。仅在 VirusAI.setVirus() 清空 GameState.activeStatuses 时失效（切换病毒用）。但目前没有关卡切换时清空 activeStatuses 的逻辑，T12 实现 startNextLevel 时要注意清空上一关病毒挂的所有状态。

**source_id 去重：** addStatus() 检查 source_id，相同来源的状态只保留最新。病毒持续效果（如 NSP1 氨基酸消耗）用 `${virus_id}_${theft_id}` 作为 source_id，防止重复叠加。

**VirusAI.setVirus() 会清空：** nullifiedCards.clear()，viralLoadFloor=0，isOverloaded=false，内部计数器重置。但不会清空 GameState.activeStatuses 中病毒挂的持续状态——T12 的 startNextLevel 需要手动清空这些。
